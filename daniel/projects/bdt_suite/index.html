The Bamboo Dumb Truck Suite (BDT Suite) is a set of webserver tools designed with personal use in mind. They are optimized for a technically knowledgeable, impatient user, who wants software to be as easy to (re)install as possible.


Presently, it consists of a nodejs webserving tool, and several Python scripts.


The python scripts cover mass downloading from DeviantArt, Newgrounds, and manga websites. There's also a script for generating sitemaps for a website.


Documentation is still being written presently, and I am modifying them to use docopt (an excellent tool for streamlining the implementation of arguments to a CLI program). When satisfactory, they will all be uploaded to the Python Package Index. The Nodejs tool is presently hosted on github, as is the sitemap generator (which happens to be the first tool uploaded to PIP in beta form).
