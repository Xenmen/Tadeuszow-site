I've been using Firefox's built-in colour options to customize my browsing for years. I've decided to switch to using the Firefox addon called Stylish, and another called Greasemonkey, both of which I'd only been using as supplements before. First, you need to have Firefox or Chrome installed, and then install the Firefox or Chrome version of the Stylish addon, and then you can install my custom CSS.

Custom Javascript for use with Greasemonkey is forthcoming.
