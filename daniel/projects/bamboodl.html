Bamboodl is a Python script for archiving imageboard threads and art/music. It is still under development and not yet ready for wider release, due to not yet having a graphical interface, but it is already stable for the sites it supports.

Currently supported sites are: 8Chan, 4Chan.

Sites for which support is planned but not yet ready: DeviantArt, Newgrounds, Tumblr, Furaffinity, Furaffinity, Inkbunny, Mangafox, Pixiv, and PHPBB based forums.
