Fresh Music News

Note that although Google Chrome is a modern browser, it does not come with a built-in RSS parser, unlike Firefox. Additionally, many Chrome RSS plugins seem to only display feed titles instead of entire descriptions, which means you won't see the full content of these feeds. I would recommend that you either use a complete RSS solution such as the Feedbro Chrome plugin, or the Mozilla Firefox browser instead of Chrome.

If you just really like using Chrome, consider installing the BBC Radio Tuner for extra news.
